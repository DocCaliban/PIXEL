Pixelcade Install Instructions RetroPie:

Refer to setup instructions at step 11 https://www.instructables.com/id/Vertical-Bartop-Arcade-With-Integrated-PIXEL-LED-D/

Summary of steps:

1. Install RetroPie on your Pi

2. Install Java 8 on your Pi

3. Create a new folder /home/pi/pixelcade

3. Copy pixelweb.jar to /home/pi/pixelcade folder

4. cd /home/pi/pixelcade

5. java -jar pixelweb.jar 

6. Copy retrogame.cfg to your Pi /boot/retrogame.cfg

7. Copy runcommand-onstart.sh and runcommand-onend.sh to your Pi /opt/retropie/configs/all/

8. Add java -jar pixelweb.jar to auto start

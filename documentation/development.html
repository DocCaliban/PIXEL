System requirements:

Here are a list of supported IDEs that can be used to develop PC-based PIXEL apps.

  • Maven 3
  • JDK 7 update 6 or latter

Supported IDEs

  • eclipse 4 (Juno)
  • Netbeans 7

eclipse

Within eclipse choose Help -> Install New Software. Then look for and install the eclipse Market Place.

Accept the default installation options and license.

Choose Help -> Eclipse Marketplace. Use m2e-wtp as a search target. Then find and install Maven integration for Eclipse WTP (Incubation)

Import existing Maven projects should be available now. Import the IOIO PC library and PIXEL PC project at the location of their respective pom.xml files.


Netbeans

Open the IOIO PC library and PIXEL PC project at the location of their respective pom.xml files.
